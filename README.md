# interactive-gauge-dashboard

Here is a Template for an interactive Dashboard with Sample Data that gets controlled by a gauge controller at the bottom of the screen that is sticky in the foreground. please try to stay as close as possible to the desired version and try to replicate the Graph and Bar groups of Data 1-5. And make it amazing. Thanks.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/interactive-gauge-dashboard.git
cd interactive-gauge-dashboard
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
